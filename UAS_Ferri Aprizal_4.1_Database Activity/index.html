Pengertian Moodle

  • Moodle merupakan salah satu paket software untuk membuat suatu pelatihan-pelatihan berbasis web dan internet yang bisa disebut sebagai Learning Management System (LMS) / Course Management System (CMS) / Virtual Learning Management Enviroment (VLE). Moodle disediakan secara gratis dan bebas digunakan karena merupakan software open source (dibawah lisensi GNU Public).

Pengertian Database Activity

  • Dengan aktifitas ini, teacher atau students dapat membuat, melihat dan mencari bank data mengenai topik apapun. Format dan struktur data yang dimasukkan hampir tidak terbatas, termasuk gambar, file, URL, nomor, dan text.
